# CIS344Project1
First individual project for CIS 344. A craft brewery database. 

This project includes the following files

1. chenDBpicture - A picture of my handwritten Chen style ER diagram
2. projectdocumentation - A reflection of how and why I selected the entities I did for my database
3. breweryDB - MySQL Workbench template for my database
4. breweryScript - SQL script for creating, using and testing my database
